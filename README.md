# 🌸 Please Forgive Me! - Cute Interactive Website 🥺

An ultra-cute, interactive, and emotional anime/cat-themed website created to say sorry in a fun way. Perfect for making up with your sister or best friend! It features premium animations, dynamic particle systems, and automated theme-based music transitions.

## ✨ Features

* **💌 Suspenseful Welcome Screen:** A sweet greeting pop-up box that smoothly bypasses modern browser autoplay restrictions.
* **🎵 Smart Audio Transition:** Plays a cute background track (`Meow MEow.mp3`) initially, which instantly pauses when "Yes" is clicked, giving way to the celebratory track (`Sistersong.mpeg`).
* **📻 Dynamic Music Control:** Real-time updates to the music player title. Shows *"I love you sister ❤️"* and *"Best sister 🎵"* right after forgiveness!
* **🏃‍♂️ Evasive 'No' Button:** An ultra-smooth, floating "No" button that randomly runs away whenever hovered over or clicked, making it impossible to refuse!
* **💥 Celebration Visuals:** Interactive canvas particles triggering full-screen firecrackers and party poppers on success.

## 🛠️ Built With

* **HTML5** & **JavaScript (ES6+)**
* **Tailwind CSS** (via CDN for ultra-smooth UI layout)
* **Canvas Confetti** (via CDN for advanced firecracker rendering)
* **FontAwesome Icons** (via CDN)
